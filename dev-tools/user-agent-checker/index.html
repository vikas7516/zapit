About User Agent Strings

A User Agent string is a text string that web browsers send to websites to identify themselves. It contains information about the browser, operating system, device, and other capabilities.

User Agent Format

User Agent strings typically follow this pattern:

Mozilla/5.0 (Platform; Security; OS-or-CPU; Localization) Gecko/GeckoDate ProductName/ProductVersion

Common Uses

  • Content Adaptation: Serve different content for mobile vs desktop
  • Feature Detection: Enable/disable features based on browser capabilities
  • Analytics: Track browser and device usage statistics
  • Security: Block or allow access based on browser type
  • Debugging: Identify browser-specific issues

Privacy Considerations

User Agent strings can be used for browser fingerprinting. Modern browsers are working to reduce the amount of identifying information in User Agent strings to improve user privacy.
