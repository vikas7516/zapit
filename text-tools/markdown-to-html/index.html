Markdown Tips

                        
    
                            
  • Headers: Use # to ###### for levels 1-6 headings.
    • 
                            
  • Lists: Start lines with -, *, or numbers like 1. for bullet or ordered lists.
    • 
                            
  • Code: Wrap inline code with backticks and multi-line blocks with triple backticks (```).
    • 
                            
  • Links & images: Format links as [title](https://...) and images as ![alt](https://...).
    • 
                            
  • Sanitization: Keep the sanitize option enabled to strip scripts and inline events from generated HTML.
